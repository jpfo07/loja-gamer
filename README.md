
# 🎮 ForgeGaming — Loja Gamer

## 🌐 GitHub Pages
🔗 **https://jpfo07.github.io/loja-gamer/**

## 📁 Estrutura de Pastas

```
loja-gamer/
├── index.html              ← Página Home
├── pages/
│   ├── portifolio.html     ← Portifólio com cards, filtros e tabela de preços
│   ├── sobre.html          ← História da marca e equipe
│   └── contato.html        ← Formulário de contato
└── src/
    ├── assets/
    │   └── img/            ← Imagens e assets visuais
    ├── css/
    │   └── estilo.css      ← Estilos globais 
    └── js/
        └── script.js       ← Lógica JS
```

---

## 🚀 Tecnologias Utilizadas

- **HTML5** — Estrutura semântica (header, nav, main, section, footer)
- **CSS3** — Variáveis CSS, Flexbox, Grid, Animações, Media Queries
- **JavaScript Vanilla** — DOM manipulation, Intersection Observer, Countdown Timer
- **Git & GitHub Pages** — Versionamento e deploy

---

## 🎨 Features

- ✅ Hero com animações de entrada e estatísticas dinâmicas
- ✅ Navbar responsiva com hamburger menu
- ✅ Cards de produtos com badges (HOT, NOVO, % OFF)
- ✅ Sistema de filtros por categoria (Portifólio)
- ✅ Countdown timer de promoção
- ✅ Tabela de preços (Planos Starter / Pro / Elite)
- ✅ Timeline da história da empresa
- ✅ Formulário de contato com validação e feedback visual
- ✅ Toast notifications
- ✅ Contadores animados com Intersection Observer
- ✅ Footer completo com redes sociais, telefone, email e copyright
